# angular-admin-lte

Admin-lte implementation for Angular 4+

## Doc & Demo
[https://mika-el.github.io/angular-admin-lte](https://mika-el.github.io/angular-admin-lte/)

## Installation

To install this library, run:

```bash
$ npm i -S angular-admin-lte admin-lte-css bootstrap-css-only@3.3.7
```
## Install demo
```bash
$ git clone git://github.com/mika-el/angular-admin-lte
$ cd angular-admin-lte
$ npm install
```

## Run demo
```bash
$ ng serve
```

## Build demo
```bash
$ ng build --prod
```

## Build dist
```bash
$ npm run build:lib
```
